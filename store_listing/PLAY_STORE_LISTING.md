## Full Description (4000 chars max)

**Folio** is the only PDF app you'll ever need. Powerful, private, and beautifully designed.

🔧 **32 Professional Tools — All Offline**
Every single feature works without an internet connection. Your documents never leave your device.

📄 **PDF Tools**
• Merge PDFs — Combine multiple files with drag-to-reorder
• Split PDF — Extract pages or split at custom ranges
• Compress PDF — Reduce file size with smart quality presets
• Rotate Pages — Fix orientation with thumbnail preview
• Reorder Pages — Drag and drop page arrangement
• Edit PDF — Add text, images, drawings, and highlights

🔄 **Convert Anything**
• Image → PDF — Photos to PDF with page size options
• PDF → Images — Export pages as JPG or PNG
• Word ↔ PDF — Convert Word documents both ways
• PowerPoint ↔ PDF — Presentation conversion
• Excel → PDF — Spreadsheet to PDF
• PDF → Text — Extract all text content
• Universal Converter — One screen for all conversions

🔒 **Security & Signing**
• Protect PDF — 128-bit AES encryption
• Unlock PDF — Remove password protection
• Watermark — Add text or image watermarks with live preview
• E-Sign — Draw, type, or upload your signature

🧠 **Smart Tools**
• Health Checker — Analyze PDF structure, metadata, and issues
• WhatsApp Shrinker — Compress to under 100KB for WhatsApp sharing

✨ **Why Folio?**
• 100% Offline — Nothing uploaded, ever. Complete privacy.
• Material 3 Design — Beautiful, modern, smooth animations
• Dark Mode — Full dark theme with automatic system detection
• Fast — Optimized for speed on any Android device
• No Bloat — Clean, focused, no unnecessary features

📱 **Works on Android 8.0+**
Designed for phones and tablets.
